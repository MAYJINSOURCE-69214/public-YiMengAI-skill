# 电商标题关键词库 v2.0

## 三段式结构

```
[产品类型 + 材质/描述] [空格] [高权重词组合] [空格] [产品名]
```

---

## 第一段模板（产品类型 + 材质/描述）

```
{type}设计{material}{type}
{type}轻奢{material}女款
{type}转运{material}女
{type}高级感{material}女手工
{type}双圈{material}手串转运
{type}天然{material}设计款
```

## 第三段：产品名处理

### 去重规则
如果第一段已包含产品类型，第三段产品名需移除末尾的类型词。

**示例：**
- 产品名：金钱召唤双圈手链
- 第一段：手链设计天然黄水晶手链（含"手链"）
- 第三段：金钱召唤双圈（移除"手链"）

### 不去重情况
如果第一段没有明确的产品类型词，第三段保留完整产品名。

## 字符计算

```
1 个中文 = 2 字符
1 个空格 = 1 字符
1 个英文字母 = 1 字符

建议总字符数 ≤ 60 字符
```

### 计算示例
```
"手链设计天然黄水晶手链 轻奢气质女款高级转运饰品 金钱召唤双圈"
= 28 + 2 + 24 + 2 + 16 = 72 字符
```

## 平台适配建议

### 淘宝
- 字符限制较宽松，可达 60-80 字符
- 可使用"招财"等词

### 抖音
- 字符建议 50-60 字符
- 避免过于迷信的词汇

### 小红书
- 字符建议 40-50 字符
- 风格偏向"轻奢"、"小众"、"设计感"
