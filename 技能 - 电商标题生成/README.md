# 技能 - 电商标题生成

一个专为 LLM/智能体设计的技能包，用于生成电商平台三段式商品标题。

## 适用平台

- 淘宝
- 抖音
- 小红书
- 其他电商平台

## 三段式结构

```
[产品类型 + 材质/描述] [空格] [高权重词组合] [空格] [产品名]
```

**示例：**
```
手链设计天然黄水晶手链 轻奢气质女款高级转运饰品 金钱召唤双圈
```

## 功能特点

- 三段式标准结构
- 产品类型必须放第一段开头
- 自动生成多样化标题
- 支持招财/转运等风格词
- 自动去重（避免重复词）
- 每次生成 5 条不同标题

## 快速使用

### 方式一：使用 Python 脚本

```bash
python scripts/generate_titles.py --product "产品名" --material "材质"
```

**示例：**
```bash
python scripts/generate_titles.py --product "金钱召唤双圈手链" --material "天然黄水晶"
```

### 方式二：直接给 LLM

将 `SKILL.md` 的内容提供给 LLM，然后输入：

```
产品名：金钱召唤双圈手链
材质：天然黄水晶
```

LLM 会根据技能说明直接生成标题。

## 注意事项

1. **产品类型必须放第一段开头**（手链/手镯/吊坠等）
2. **第三段产品名要去重**（如果第一段已有"手链"，产品名去掉"手链"）
3. **字符数建议≤60**（部分平台可放宽）
4. **根据平台调整风格词**（淘宝可用招财，小红书偏向轻奢小众）
